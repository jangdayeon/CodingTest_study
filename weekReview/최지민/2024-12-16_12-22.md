**2. 삼각 달팽이 (68645)**
- 규칙성을 찾고 **방향 벡터**를 사용해 풀이하는 문제

- 방향 이동 문제에서 자주 쓰이는 **dx, dy 테크닉**:
    ```
    dx = [1, 0, -1, 0]
    dy = [0, -1, 0, 1]
    ```
    위처럼 설정하면, 0, 1, 2, 3의 인덱스를 이용해 각 방향으로 한 번씩 이동 가능
